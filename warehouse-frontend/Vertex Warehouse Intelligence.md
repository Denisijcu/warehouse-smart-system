# Vertex Warehouse Intelligence (VWI) 🚀

Plataforma de gestión de almacén de nueva generación, desarrollada bajo los estándares de **Vertex Coders LLC** para optimizar operaciones logísticas mediante Inteligencia Ofensiva y automatización inteligente.

## 🛠 Stack Tecnológico

* **Frontend:** Angular 19 (Standalone Components, Signals).
* **Estilos:** Tailwind CSS (Dark Mode corporativo).
* **Arquitectura:** State Management reactivo, Lazy Loading de módulos, Auth Guards para seguridad.
* **Integración AI:** Módulo de asistencia local (`AiAssistant`) para consulta de datos en tiempo real.

## 🏗 Estructura del Sistema

El sistema ha sido diseñado modularmente para garantizar escalabilidad y mantenimiento:

| Módulo | Descripción |
| --- | --- |
| **Dashboard** | Vista centralizada de métricas operativas. |
| **Inventory** | Gestión de productos con persistencia de estado. |
| **Movements** | Seguimiento de flujo logístico y trazabilidad. |
| **Suppliers** | Gestión relacional de proveedores. |
| **Reports** | Análisis predictivo y financiero de inventario. |
| **AI Assistant** | Copiloto basado en contexto local para consultas rápidas. |
| **Alertas** | Monitoreo proactivo de stock crítico (UI reactiva). |
| **Configuración** | Control de identidad y endpoints de OracleAI. |

## 🛡 Características Principales

* **Computational Sovereignty:** Interfaz diseñada para ejecutarse con modelos locales, garantizando la privacidad de los datos operativos.
* **Offensive Reconnaissance Ready:** Auditoría de seguridad integrada bajo los estándares OWASP para proteger la integridad de los datos de Vertex Coders.
* **UI/UX de Élite:** Diseño consistente con *custom-scrollbar*, gradientes corporativos y una navegación fluida que asegura cero fricción operativa.

## 🚀 Despliegue Técnico

El proyecto está configurado para despliegue en entornos Docker/VirtualBox, priorizando el *hardening* de seguridad en cada componente.

> **Nota de CEO:** Este sistema es el core de la plataforma OracleAI. La automatización es el objetivo; la seguridad es la base.
